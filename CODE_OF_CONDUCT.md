# Code of Conduct

This project adopts the **Contributor Covenant** (version 2.1) as its code of conduct. The full text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Summary

In short, contributors and maintainers commit to making participation in this project a respectful, professional experience focused on technical merit. Behaviour that disrupts the technical work of the community will be addressed by the maintainers.

## Reporting

To report a concern, contact the maintainers at `security@vernammail.com`. Reports will be handled confidentially.

## Scope

This Code of Conduct applies to all project spaces (issues, pull requests, discussions, mailing lists, and any official communication channels) and to representations of the project in public contexts.
